Afin d’éviter à tout le monde de créer un compte sur Kaggle, les données peuvent être téléchargées à partir du répertoire du cours: https://github.com/FAS1002/A22/blob/main/assets/fichiers/olympic_archive.zip

Téléchargement du fichier

Pour télécharger le dossier compressé (.zip), il faut le lien direct et il est possible de l’obtenir en copiant le lien du bouton “Download”.

Code 
URL <- "https://github.com/FAS1002/A22/raw/main/assets/fichiers/olympic_archive.zip"

base_path <- path("data", "raw")

fname <- paste(today(),
               "olympic-archive.zip", 
               sep = "_")
fpath <- path(base_path,
              fname)

download.file(url = URL, 
              destfile = fpath)

      exdir = base_path)
